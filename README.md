# GODOT-3D-INFINITE-RUNNER-GAME
# 3D Endless Running Game (Godot Engine)

## Description
This is a 3D endless running game developed using the Godot Engine, inspired by Subway Surfers. In this game, the player continuously runs forward, avoids obstacles, and collects coins along the way. The game also displays the player’s score and high score to track performance.

This project was developed along with my partner **Eman Zahid** as part of our game development practice to improve our understanding of 3D game mechanics, player controls, and basic game design using the Godot Engine.

## Features
- Endless running gameplay
- Coin collection system
- Score and high score tracking
- Smooth player movement controls
- Simple and engaging 3D environment

## Technologies Used
- Godot Engine
- GDScript
- 3D Game Development Concepts

## How to Play
- Use controls to move the player left/right/jump
- Avoid obstacles
- Collect coins to increase score
- Try to beat your high score

## Project Goal
The goal of this project was to learn and implement basic 3D game development concepts such as player movement, collision detection, scoring system, and level design using Godot Engine.
